Welcome to Chimèra

Chimèra is an actively developed open source MMORPG. It differs from all other MMORPGs because anyone can create an instance and add it to the Chimèra Federation as happens in the "fediverse", about social networks like Mastodon.

You can see Chimèra as something between second life and an RPG, placed on a persistent virtual universe. Want to know more? Give a read to our wiki!

We need you!

At this point, we need all the help we can get! The community is very italo-centric since it was born in Italy, though we welcome anyone. Are you a designer, a developer or a sound artist? Come join us!

Our development is based on github and we meet on discord.
